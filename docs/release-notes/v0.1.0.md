**TypeX 0.1.0 — Beta**

The first public build. Windows only for now.

### Install

Download **[TypeX_0.1.0_x64_en-US.msi](https://github.com/Xyhlo/typex/releases/latest/download/TypeX_0.1.0_x64_en-US.msi)** and run it. The MSI bundles Pandoc so 40+ format import/export works out of the box — nothing else to install.

- **Installer size**: 44 MB download, ~225 MB on disk (mostly Pandoc).
- **Requires**: Windows 10 or 11, x64. WebView2 bootstraps automatically on first run if you don't already have it.
- **File associations**: registers TypeX as an "Open with" option for 17 format groups (`.md`, `.markdown`, `.mdx`, `.txt`, `.docx`, `.odt`, `.rtf`, `.html`, `.htm`, `.epub`, `.rst`, `.adoc`, `.asciidoc`, `.tex`, `.latex`, `.org`, `.textile`, `.wiki`, `.mediawiki`, `.ipynb`, `.fb2`, `.opml`).
- **Single-instance**: double-clicking another file routes into the running window instead of spawning a second copy.

### What's in it

- Live WYSIWYG markdown editing (Milkdown / ProseMirror) with markdown-aware paste.
- Two hand-tuned themes: **Obsidian Ink** (dark) and **Ivory Paper** (light).
- Real menu bar — File / Edit / View / Insert / Format / Tools / Help — with Alt-accelerators, submenus, checkmarks, shortcut hints.
- Command palette (`Ctrl+K`), find + replace via CSS Highlight API (`Ctrl+F` / `Ctrl+H`).
- Tabs with dirty indicator, close-others/close-all, session restore.
- File tree sidebar, outline panel generated from headings.
- 40+ format import/export via bundled Pandoc 3.9.
- Syntax-highlighted code blocks via Lowlight + ProseMirror decorations.
- Autosave, typewriter mode, focus mode, reading-width toggle (vertical/horizontal), editor font toggle (sans/serif), zoom.
- Preferences modal (`Ctrl+,`) with the knobs most people actually want.

### Not in this release

Reminders and JSX rendering are on the roadmap but don't ship in 0.1.0. macOS and Linux builds come after Windows stabilizes.
