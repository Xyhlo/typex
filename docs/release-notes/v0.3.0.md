**TypeX 0.3.0 — Editor craft + git-native + live streams**

The biggest release since the initial beta. Two full roadmap phases ship together: Phase 1 (editor craft) and Phase 2 (git-native). Plus a multi-root workspace, live-streaming external writes, and a clone dialog.

### New — editor craft

- **Tri-mode view.** WYSIWYG ↔ raw markdown ↔ split (raw left, preview right). `Ctrl+/` toggles raw, `Ctrl+Shift+/` toggles split, or click the mode button in the status bar to cycle.
- **Wikilinks.** Type `[[Some Note]]` in any document — it renders as a clickable inline link. `Ctrl+click` navigates. Broken links (no matching file in the vault) show as a dashed amber underline, not a spellcheck squiggle.
- **Tags.** `#tag` anywhere in prose → a dedicated Tags sidebar panel lists every tag in the vault with counts, click to expand to its files.
- **Backlinks.** A Backlinks sidebar panel shows every file that links to the currently-open document via `[[...]]`.
- **Properties.** Dedicated sidebar panel renders YAML frontmatter as key/value rows with chip-style arrays.
- **Outline polish.** Duplicate headings are click-to-scroll correctly (index-based, not text-match). Active-heading highlights as you scroll. `~~~` fences recognized alongside ```` ``` ````. Frontmatter is skipped so `# Title` inside it doesn't become a heading.
- **Smarter paste.** Shebang detection extended to `ts-node`, `deno`, `fish`, `awk`, `tclsh`, `Rscript`, `php`, `lua`. Debug logging is gated behind `window.__typexPasteDebug = true`.

### New — multi-root workspace

- **Open multiple folders at once.** Each stacks in the Files panel with its own collapsible header and a hover-revealed `×` to remove.
- **Recent folders dropdown.** Clock icon in the Files panel opens a menu of recently-used folders, with currently-open ones shown as disabled-checked for clarity.
- **Aggregates across roots.** Tags, backlinks, outline, and properties all span every open root — a vault of three repos behaves like one.
- **Ghost-root detection.** Session restore filters out folders that no longer exist on disk with a toast, so stale roots don't haunt the sidebar forever.
- **Wikilink collision warnings.** Two files with the same basename across different roots get a devtools warning so you know backlinks may be ambiguous.

### New — git integration (no terminal required)

- **Status bar sync pill.** Shows branch, clean/dirty, `↑ahead / ↓behind`, merging, detached, `upstream gone`, `empty` (fresh `git init`), and git errors. Click to refresh.
- **Gutter marks in raw mode.** Green bars for added lines, amber for modified, red triangles where content was deleted above. Live against HEAD, respects fenced code blocks.
- **Blame on Ctrl-click.** Click any line in raw mode with Ctrl held → toast shows `sha · author · relative-time · "commit message"`. Uncommitted lines say so.
- **External-change detection.** When another process writes to a file you have open: silent reload if your tab is clean, "Keep mine / Reload from disk" modal if dirty. Deleted-on-disk warning when an open file is removed outside TypeX.
- **Autocommit on save** (opt-in, debounced 15s). Commit messages are workspace-relative, deduped across paths: `Update docs/intro.md, docs/setup.md`.
- **Autopush after commit** (opt-in). Uses your system git credential helper — TypeX never stores credentials. Push failures surface a readable hint ("credentials not configured — run `git push` once in a terminal") and auto-retry in 30 s.
- **Pull on focus** (opt-in, `--ff-only`). Skipped with a toast when the working tree has uncommitted changes; rate-limited so alt-tabbing doesn't spam.
- **Clone a git repository.** `File → Clone git repository…` or the `git.clone` command palette entry. Works with any git URL — GitHub, GitLab, SSH, self-hosted. URL allowlist rejects flag-injection. Missing credentials fail fast instead of hanging.
- **`git push` / `git pull` / `git commit` / `git refresh status`** palette commands for everything the autosync loop can do manually.

### New — live streaming from external writes

When an AI agent (or any rapid-write tool) modifies a file you have open, TypeX doesn't snap to the final state — it types the new content in visibly:

- **Blinking caret** at the growing edge, accent-colored with a small glow.
- **Content types in over ~400ms** — 25-ish rAF ticks gradually apply the new content so you see characters materialize rather than jump.
- **Accent-colored editor border** pulses while any stream is in flight.
- **Tab pulse** — streaming tabs wash accent-colored and gain a pulsing dot.
- **`LIVE` pill** in the status bar, unmistakable.
- **Per-block flash** with a 3px accent left-bar after the stream settles.
- **Size guard** — docs over 60k chars or deltas under 24 chars skip animation. Tab switches mid-stream cancel cleanly.

All animations are gated by the `Animate external edits` pref, on by default. `Live-reload external writes` is also on by default so streams apply through instead of popping a modal per chunk.

### Fixed

- **Stream `onDone` leak** — a superseded stream-apply no longer strands its promise, which could leave the editor unable to accept input.
- **Tab-switch mid-stream corruption** — pending stream ticks are cancelled so they can't write into the new tab.
- **Flash-timer leak across tab churn** — flash `setTimeout`s are now tracked per-view and cancelled on destroy.
- **Viewport yank at stream completion** — scroll position is no longer forced back to the pre-stream position after a long append.
- **Deleted-file purge** — deleting a file externally now evicts it from the vault index so its backlinks and tags clear.
- **Second-file conflict modal drop** — conflict modals now queue per path instead of silently dropping.
- **Silent autopull-skip** — when the working tree is dirty, the skip is now announced once per minute instead of vanishing.
- **Credential-prompt hangs** — `GCM_INTERACTIVE=Never` + `SSH_ASKPASS_REQUIRE=never` + `GIT_TERMINAL_PROMPT=0` + `stdin=null` on every git spawn means auth failures return immediately with a readable message instead of hanging the app.
- **URL injection hardening** — `is_safe_git_url` now rejects embedded whitespace, flag-like paths after `@host:`, and SSH short-forms that would let `git`'s transport helper parse `--upload-pack=` out of the URL itself.
- **Multi-root race** — concurrent add/close folder operations are serialized through a promise chain so `setVaultRoot` and `watchRoots` can't land out of order.
- **Raw-mode split** now has raw on the left, preview on the right (was reversed). Divider only shows in split mode.

### Install

Download **[TypeX-latest-x64.msi](https://github.com/Xyhlo/typex/releases/latest/download/TypeX-latest-x64.msi)** or the versioned **[TypeX_0.3.0_x64_en-US.msi](https://github.com/Xyhlo/typex/releases/latest/download/TypeX_0.3.0_x64_en-US.msi)** and run it.

- **Size:** ~45 MB download, ~225 MB on disk (most of it Pandoc 3.9).
- **Requires:** Windows 10 or 11, x64. WebView2 auto-installs on first run if needed.
- **Upgrades cleanly in place** from 0.1.x.
