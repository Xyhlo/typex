TypeX
Design System · by Amptis
v0.1.0 · Windows · MIT

A Markdown editor that puts typography first.

This is the living source of truth for TypeX — two hand-tuned themes sharing identical semantic tokens, an em-based heading scale, and a chrome that recedes so the writing can read.

Colors
Obsidian Ink (dark) and Ivory Paper (light) — same semantic vars, different literals. The indigo accent is #8b7cff on dark and #5b4de0 on light so the hue reads as the same color in both modes.
Type
Inter for UI and editor body, JetBrains Mono for code, optional Iowan Old Style / Palatino serif mode. Heading tracking -0.014em, hero -0.02em. Em-based scale from h1 down.
Grid & motion
4 px spacing. Radii 3 / 5 / 8 / 12 / 16 + pill. Motion 90 / 160 / 220 / 320 / 480 ms on cubic-bezier(0.2, 0, 0, 1). prefers-reduced-motion zeroes them.
Voice
Quiet, considered, a little literary. Sentence case. Third person for the product, second person in help text. Em-dashes pair claim with reason. No emoji, no exclamation points.
